# Git for Windows
Git for Windows是Git官方提供的Windows版本，包括命令行和图形界面。

[git-for-windows](https://git-for-windows.github.io/)


Git for Windows focuses on offering a lightweight, native set of tools 
that bring the full feature set of the Git SCM to Windows while 
providing appropriate user interfaces for experienced Git users and novices alike.
